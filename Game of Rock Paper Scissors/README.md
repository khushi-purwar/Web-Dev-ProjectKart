


##  Rock-Paper-Scissor Game

## About the Project

A web app based game of rock paper and scissors.

## Rules

If the player wins, they gain 1 point that add up in their score.

## Rules

- Paper beats Rock
- Rock beats Scissors
- Scissors beats Paper

## Tech Stacks used

- HTML
- CSS
- JS
